# Esports DBMS GUI

This folder contains the static frontend for presenting the DBMS project. Open [index.html](index.html) in a browser, or open the root [../index.html](../index.html) and it will redirect here.

No server or installation is required.

## Files

- [index.html](index.html) - Main GUI page.
- [style.css](style.css) - Layout and visual styling.
- [app.js](app.js) - Data, tabs, filters, and query viewer behavior.
